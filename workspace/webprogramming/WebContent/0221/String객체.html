charAt()메서드

charAt(index) : index위치에있는 문자를 리턴한다

indexOf()메서드

indexOf() : 부분문자열의 위치를 검색한다 indexOf(param1, param2) : param1 - 문자열 param2 - 검색시작인덱스

concat()메서드

concat(s1, s2) : s1문자열과 s2문자열을 연결한다, +연산과 동일

trim()메서드

trim() : 앞뒤 공백을 제거한다

ToLowerCase(), toUpperCase()

ToLowerCase() : 소문자로 변경 toUpperCase() : 대문자로 변경

substring()메서드

substring(from, to) : 두 위치 사이의 부분 문자열을 추출한다 to를 생략하면 뒤쪽 모든 문자열을 출력한다. to 미만까지 출력한다.

slice() 메소드

slice(start,end) : start위치부터 end위치까지 부분 문자열을 추출한다. substring()과의 차이점은 slice()는 음수값을 파라미터로 적용하여 문자열을 뒤에서 부터 출력할 수 있다. substring()은 음수 사용 불가능

split() 메서드

split() : 구분자를 기준으로 문자열을 분리한다.
